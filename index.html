HTML

  • Head elements are not visible to end user but allow the browser to understand how web application should render
  • Body elements are visible to end user and describe how page should render
  • HTML sets up structure of the page, it is the foundation
  • H1 tag describes the largest header on the page and should only be used once per page

CSS

  • CSS or cascading style sheets apply a style to the elements
  • CSS can apply things like color to your content
  • There are three ways to approach CSS inline style, internal style sheets, and external style sheets
  • using the link tag, you can link an external sheet to your html file and href tag will indicate where the sheet lives

Git

  • Git is a free open source version control platform that developers can collaborate in without the fear of overwriting or disrupting each other's work
  • Version control is critical for software developers as you may want to revert to previous versions or double check work from prior iterations
  • git status: checks what branch we are currently on
  • git checkout -b branch-name: creates a new branch and switches to it

JavaScript

  • JavaScript allows you to dictate how the end user will interact with your web application
  • A variable is a named container that allows us to store data in our code.
  • Control flow is the order in which a computer executes code in a script.
  • Functions are the building blocks of JavaScript
  • JavaScript reads code from top to bottom also known as, "Control Flow"
